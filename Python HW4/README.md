## Рекомендації до виконання

У **4-му завданні** обов’язково мають бути реалізовані **три функції**:

### 1. `simple_prime_search(n)`
Функція реалізує простий спосіб перевірки, чи є число простим, шляхом перебору всіх можливих дільників.

- Простий та наочний підхід
- Низька ефективність для великих значень `n`

---

### 2. `sieve_eratosthenes(n)`
Функція реалізує алгоритм **решета Ератосфена**.

- Більш ефективний метод пошуку простих чисел
- Працює шляхом виключення чисел, кратних простим
- Повертає список простих чисел до заданого ліміту `n`

---

### 3. `measure_time(func, *args, **kwargs)`
Функція для вимірювання часу виконання іншої функції.

#### Логіка роботи:
- `start = time.time()` — фіксація початку виконання
- `result = func(*args, **kwargs)` — виклик переданої функції
- `end_time = time.time()` — фіксація завершення виконання
- Виведення часу виконання у форматі:  
  `Time of the <function_name> function: X.XXXXXXXXX sec.`
- `return result` — повернення результату виконання функції

#### Призначення:
- `*args` та `**kwargs` дозволяють працювати з будь-якою функцією незалежно від її сигнатури
- `measure_time` повертає результат виконання `func` та виводить час її виконання
