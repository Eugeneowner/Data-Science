## Завдання

### 1. Завантаження набору даних
- Перевірити наявність файлу `movies.csv` у поточному каталозі.
- Якщо файл відсутній:
  - виконати HTTP-запит за вказаною URL-адресою;
  - перевірити, що код відповіді дорівнює **200 (OK)**;
  - зберегти отримані дані у файл `movies.csv`.
- Зчитати файл у `DataFrame` за допомогою `pandas`.
- Видалити дублікати рядків.

---

### 2. Аналіз структури та типів даних
Виконати аналіз структури набору даних:
- вивести список усіх стовпців;
- визначити типи даних кожного стовпця;
- переглянути загальну інформацію про `DataFrame`;
- отримати описові статистики числових полів.

Результат повинен дати розуміння:
- які дані містяться у наборі;
- які поля є числовими, текстовими або категоріальними;
- наявність пропущених значень.

---

### 3. Кількість фільмів у наборі даних
Визначити загальну кількість фільмів **щонайменше п’ятьма різними способами**, зокрема:
1. через унікальні значення у стовпці з назвою фільму;
2. через загальну кількість рядків у `DataFrame`;
3. через підрахунок ненульових значень;
4. через підрахунок частоти значень;
5. через атрибут `shape`.

---

### 4. Кількість фільмів за роками
Визначити, скільки фільмів припадає на кожний рік випуску.  
Застосувати **не менше п’яти різних підходів**, зокрема:
- підрахунок частот значень;
- групування з агрегацією;
- використання `count()` після `groupby`;
- вибір потрібних стовпців після групування;
- підрахунок у циклі з фільтрацією за роком.

Результат має бути представлений у зручному для аналізу вигляді.

---

### 5. Найменш і найбільш прибуткові фільми
Показати детальну інформацію про:
- **найприбутковіший фільм**;
- **найменш прибутковий фільм**.

Для цього:
- знайти максимальне та мінімальне значення показника прибутковості;
- відфільтрувати відповідні рядки;
- використати альтернативний підхід через `loc` та `isin()`.

---

### 6. Узгодження значень у стовпці «Жанр»
- Вивести список унікальних жанрів.
- Виявити непослідовні або некоректні значення.
- Виправити їх за допомогою методу заміни.
- Повторно перевірити унікальні значення жанрів після виправлення.

---

### 7. Топ-10 комедій за кількістю глядачів
- Відібрати фільми жанру **Comedy**.
- Відсортувати їх за кількістю глядачів у порядку спадання.
- Обрати **10 найкращих**.
- Залишити лише такі поля:
  - назва фільму;
  - рік;
  - студія.
- Зберегти результат у **новий CSV-файл**.

Допускається альтернативна реалізація з використанням `loc` та умов.

---
